# Foundation

The Koa setup is inspired by https://github.com/rethinkdb/rethinkdb-example-nodejs/tree/master/todo-angular-koa

React/Redux/Middleware from http://redux.js.org/docs/advanced/ExampleRedditAPI.html

Tech used:
* Node.js
 * Koa
 * RethinkDB
* React
* Redux
